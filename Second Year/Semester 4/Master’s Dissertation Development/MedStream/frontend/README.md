# MedStream Frontend

This is the React/Vite frontend for MedStream.

Most project documentation lives in the root `README.md`, including setup, services, environment variables, app routes, and architecture.

## Local Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

The frontend expects:

```env
VITE_API_BASE_URL=http://localhost:8000
VITE_WS_URL=ws://localhost:8000/ws
```
